### `interface` - 接口

接口是通过 interface 关键字来定义的。接口不是类，但是，使用接口可以限定实现它的类必须实现哪些方法。接口可以理解为一种协议，或者说标准，它只声明实现哪些函数，而这些函数功能的具体实现，交给实现（implements）它的类来完成。
